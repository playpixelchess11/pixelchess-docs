# PixelChess Docs

Welcome to the official documentation for **PixelChess** — a pixel-art chess game on Base network where you wager real ETH against other players.

## Overview

PixelChess is a **Player vs Player** on-chain chess game. Both players stake ETH, play chess peer-to-peer, and the winner receives **1.9× their wager** automatically via smart contract on Base.

- No house bankroll
- No sign-up required
- Fully on-chain settlement
- Peer-to-peer gameplay (no server)
